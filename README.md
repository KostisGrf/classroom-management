# 🎓 Classroom Management System

A full-stack project for managing a classroom environment including **subjects, departments, classes, faculty, and enrollments**.

Built as a learning project to explore modern full-stack development and AI-assisted workflows.

---

## 🚀 Stack

### Backend
- Node.js + Express
- TypeScript
- Drizzle ORM
- PostgreSQL (Neon)
- REST API

### Frontend
- React
- Refine.dev
- TypeScript
- Tailwind CSS
- shadcn/ui

### AI & Tools
- GitHub Copilot
- Refine AI
- CodeRabbit (PR reviews)

---

## 🧠 Learning Outcomes

- Full-stack architecture (frontend ↔ backend)
- REST API design + pagination + filtering
- Relational database design (foreign keys)
- Refine data provider system
- AI-assisted development workflow
- Debugging complex React re-fetch issues

---

## 📊 Features

- Departments management
- Subjects with department relation
- Classes system
- Faculty management
- Enrollments tracking
- Server-side filtering & pagination
